# SGT_Area
สคริปสำหรับการสร้างฟอร์มค้นหาพื้นที่ในระบบการ โดยสามารถค้นหาได้ตั้งแต่ระดับภูมิภาค ถึงระดับเขต

ฟังชั่นต่าง ๆ ใน Script นี้

getRegion() 
  - จะ return Array ของชื่อและตำแหน่งที่ใช้อ้างอิงไปยังภูมิภาคนั้น ๆ ออกมา
  - ตัวอย่างของข้อมูลที่ return ออกมา
      [
        {
          name: "มหานคร 1",
          pos: 0
        },
        // ข้อมูลภูมิภาคอื่น ๆ
      ]

getZone(rg_pos)
  - จะทำการรับตำแหน่งที่ใช้อ้างอิงไปยังภูมิภาคเข้ามาเพื่อใช้ในการอ้างอิง
  - และจะ return Array ของชื่อและตำแหน่งที่ใช้อ้างอิงไปยังรวมภาค ที่อยู่ในภูมิภาคนั้น ๆ ออกมา
  - ตัวอย่างของข้อมูลที่ return ออกมา
      [
        {
          name: "กรุงเทพ 1",
          pos: 0
        },
        // ข้อมูลรวมภาคอื่น ๆ
      ]
    
getSector(rg_pos, zone_pos)
  - จะทำการรับตำแหน่งที่ใช้อ้างอิงไปยังภูมิภาคและรวมภาคเข้ามาเพื่อใช้ในการอ้างอิง
  - และจะ return Array ของชื่อและตำแหน่งที่ใช้อ้างอิงไปยังภาค ที่อยู่ในรวมภาคนั้น ๆ ออกมา
  - ตัวอย่างของข้อมูลที่ return ออกมา
      [
        {
          name: "นนทบุรี",
          pos: 0
        },
        // ข้อมูลภาคอื่น ๆ
      ]

getDistrict(rg_pos, zone_pos, sec_pos)
  - จะทำการรับตำแหน่งที่ใช้อ้างอิงไปยังภูมิภาค รวมภาค และภาคเข้ามาเพื่อใช้ในการอ้างอิง
  - และจะ return Array ของชื่อและตำแหน่งที่ใช้อ้างอิงไปยังเขต ที่อยู่ในภาคนั้น ๆ ออกมา
  - ตัวอย่างของข้อมูลที่ return ออกมา
      [
        {
          name: "นนทบุรี",
          pos: 0
        },
        // ข้อมูลเขตอื่น ๆ
      ]
